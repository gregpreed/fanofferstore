WooCommerce Facebook Tab.  This plugin lets you embed your WooCommerce products into your Facebook Page.

== Requirements ==

 - A website with SSL enabled, this is a Facebook requirement.
 - A Facebook Page, see here to create one: https://www.facebook.com/pages/create.php

== Installation ==

1. Upload the folder 'woocommerce-facebook-tab' to the '/wp-content/plugins/' directory

2. Activate 'WooCommerce Facebook Tab' through the 'Plugins' menu in WordPress

== Usage ==

1. Go to the 'WooCommerce' > 'Facebook' menu in Wordpress.

2. Follow the instructions on screen for setting up your Facebook page.
