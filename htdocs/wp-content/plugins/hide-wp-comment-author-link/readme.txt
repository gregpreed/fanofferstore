Use Hide Comment Author Link plugin and easily disable comment author url from your WordPress site.

== Description ==

Do you want to hide or disable comment author link in WordPress? Hide Comment Author Link plugin removes all hyperlinks from comment author name. This plugin will strip all links from commenter names wherever they may appear throughout your site's theme. 

Just simply install Hide WordPress Comment Author Link plugin and it will be automatically disable your all comment author url.

If you have any questions then ask to <strong><a href="http://freebloggingtricks.com/hide-comment-author-link/">support blog</a></strong>.

Important Link: <a href="http://freebloggingtricks.com/">FreeBloggingTricks.com</a>

== Installation ==

1. Upload the `hide-comment-author-link` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You’re done!
