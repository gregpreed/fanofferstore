== Description ==

This plugins adds a new visitor condition that allows you to target ads by browser language.

This feature is already included in Advanced Ads Pro (https://wpadvancedads.com/add-ons/advanced-ads-pro/)

== Installation ==

The Browser Language plugin is based on the free Advanced Ads plugin, a simple and powerful ad management solution for WordPress. Before using this plugin download, install and activate Advanced Ads for free from http://wordpress.org/plugins/advanced-ads/.
